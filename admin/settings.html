Firebase Setup Checklist

  1. Create Firestore Database — Open Firestore → click Create database → pick a location → Enable
  2. Apply the security rules below (Rules tab → Paste → Publish)
  3. Enable Authentication → Email/Password sign-in
  4. Create your admin account in Authentication → Users
  5. Enable Storage for product images
  6. Copy your web app config into firebase/config.js
  7. Set demoMode: false in assets/js/config.js
  8. Deploy to GitHub Pages (see README.md)

Firestore Security Rules

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /products/{id} {
      allow read: if true;
      allow write: if request.auth != null;
    }
    match /categories/{id} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
